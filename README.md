# Annota Website

This repository showcases the official website for **Annota**, a privacy-focused digital mind designed to keep your thoughts, notes, and tasks fully secured through a local-first architecture.

## 🚀 Overview

Annota is built for users who prioritize data sovereignty and speed. By keeping your data local and applying end-to-end encryption, Annota ensures that your "digital mind" belongs solely to you, while still offering seamless synchronization across all your devices.

## ✨ Key Features

- **Local-First Design**: Lightning-fast performance and full offline capability.
- **E2E Encryption**: Zero-knowledge security for your notes, tasks, and assets.
- **Multi-Platform Support**: Available for macOS, Windows, Linux, Android, and iOS.
- **Cloud Sync & Recovery**: Optional secure synchronization for multi-device workflows.
- **Rich Editor**: Support for LaTeX, Code Blocks, Dynamic Tables, and more.

## 🛠️ Website Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)



*Note: This repository is the frontend showcase for the Annota platform.*
